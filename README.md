# Wiki 库

基于 SpringBoot3.x + Mybatis Plus3.5.x 开发的知识库项目(服务端完结)

- 数据增长趋势
- 富文本存储

## 环境准备

- MySQL 8.*/5.*
- Redis
- JDK17+
- Gradle
